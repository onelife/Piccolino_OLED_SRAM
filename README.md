# Piccolino OLED SRAM Library With Fonts Support #

## Fonts ##

* ASCII 16x16
* ASCII 13x12
* HZK 16x16 (Can only be loaded from SD card)
* HZK 12x12 (Can only be loaded from SD card)

## Config (font_config.h) ##

* FONT_ON_SD

  Load font data (*.fnt) from SD card root directory

* USE_FONT_SIZE_16
* USE_FONT_SIZE_12
